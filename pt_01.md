# Практическое занятие №07: Генерация текста с помощью LLM

**Модуль:** M4 | **Неделя:** 14

---

## БАЗОВЫЙ УРОВЕНЬ

**Индикатор:** ПК-14.1 LLM-5.1 (базовый)
**Цель:** Освоить техники взаимодействия с LLM через API.

### Задачи:
1. Подключиться к API языковой модели
2. Составить 5 различных промптов
3. Проанализировать качество ответов

### Ход выполнения:
- (1 ч) Знакомство с API, базовые промпты
- (2 ч) Серия запросов: суммаризация, перевод, генерация
- (1 ч) Анализ качества, итеративное улучшение

### Ожидаемый результат:
Навыки взаимодействия с LLM, 5+ промптов.

---

## СРЕДНИЙ УРОВЕНЬ

**Индикатор:** ПК-14.4 LLM-5.4 (средний)
**Цель:** Применить продвинутые техники prompt engineering.

### Задачи:
1. Реализовать Chain-of-Thought prompting
2. Применить Few-shot learning (3-5 примеров)
3. Сравнить Zero-shot vs Few-shot vs CoT
4. Оформить результаты как benchmark

### Ход выполнения:
- (1 ч) CoT: пошаговое рассуждение
- (1 ч) Few-shot: 3-5 примеров в промпте
- (1 ч) Сравнительный эксперимент
- (1 ч) Оформление benchmark

### Ожидаемый результат:
Benchmark трёх техник prompting.

---

## ПРОДВИНУТЫЙ УРОВЕНЬ

**Индикатор:** ПК-14.5 LLM-5.6 (продвинутый)
**Цель:** Разработать систему автоматической генерации отчётов.

### Задачи:
1. Создать pipeline: данные -> промпт -> генерация -> постобработка
2. Реализовать валидацию输出 (JSON schema)
3. Добавить retry-логику при ошибках
4. Оценить стоимость и latency

### Ход выполнения:
- (1.5 ч) Архитектура pipeline
- (1 ч) Валидация и retry
- (1 ч) Тестирование на 100 примерах
- (0.5 ч) Оценка стоимости

### Ожидаемый результат:
Работающий pipeline генерации отчётов.
