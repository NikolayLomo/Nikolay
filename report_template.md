# Практическое занятие №06: Работа с моделями на основе Трансформера

**Модуль:** M4 | **Неделя:** 12

---

## БАЗОВЫЙ УРОВЕНЬ

**Индикатор:** ПК-14.1 LLM-5.1 (базовый)
**Цель:** Освоить библиотеку Hugging Face Transformers.

### Задачи:
1. Загрузить предобученную модель и токенизатор
2. Выполнить Masked Language Modeling (BERT)
3. Выполнить генерацию текста (GPT-2)

### Ход выполнения:
- (1 ч) Установка transformers, токенизация текста
- (1.5 ч) BERT: fill-mask pipeline
- (1.5 ч) GPT-2: text-generation pipeline

### Ожидаемый результат:
Применение двух моделей Трансформера для NLP.

---

## СРЕДНИЙ УРОВЕНЬ

**Индикатор:** ПК-14.4 LLM-5.4 (средний)
**Цель:** Исследовать влияние параметров генерации.

### Задачи:
1. Сравнить greedy, beam, top-k, top-p сэмплирование
2. Исследовать влияние temperature
3. Проанализировать attention-веса
4. Создать сравнительную таблицу методов

### Ход выполнения:
- (1 ч) Четыре метода сэмплирования
- (1 ч) Temperature: 0.1, 0.5, 1.0, 2.0
- (1 ч) Визуализация attention
- (1 ч) Сравнительная таблица

### Ожидаемый результат:
Анализ параметров генерации с примерами.

---

## ПРОДВИНУТЫЙ УРОВЕНЬ

**Индикатор:** ПК-14.5 LLM-5.6 (продвинутый)
**Цель:** Fine-tune BERT для классификации текстов.

### Задачи:
1. Подготовить датасет для классификации
2. Fine-tune BERT на 3 эпохи
3. Оценить accuracy, F1, confusion matrix
4. Сравнить с zero-shot классификацией

### Ход выполнения:
- (1 ч) Подготовка данных, токенизация
- (1.5 ч) Fine-tuning: Trainer API
- (1 ч) Оценка метрик
- (0.5 ч) Сравнение с zero-shot

### Ожидаемый результат:
Fine-tuned BERT, сравнение с zero-shot.
